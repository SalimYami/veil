🔒 Built VEIL — a zero-knowledge cloud storage system where the server is completely blind to your data.

AES-256-GCM encryption runs in your browser via WebCrypto API.
The key is derived from your password using Argon2id (64MB WASM).

Tech stack:
→ React 19 + TypeScript (client-side crypto)
