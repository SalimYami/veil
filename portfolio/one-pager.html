🔐 Zero-Knowledge Crypto Model

Key derivation flow (browser-only):

password + email → Argon2id WASM → 32B master key
master[:16] → SHA-256 → authKey (server)
master[16:] → RAM only → encryptionKey 🔐

File encryption:

plaintext + random IV(12B) → AES-256-GCM → ciphertext+tag

⚠️ Password loss = permanent data loss (by design — true zero-knowledge)
